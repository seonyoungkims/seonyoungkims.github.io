Project Overview

Addressed the critical need for automated quality control in semiconductor Chemical Mechanical Planarization (CMP) processes. Developed a dual-track deep learning system: a computer vision framework for identifying defective wafers and a time-series anomaly detection model for monitoring equipment sensor streams.

Achievements

  • Defect Wafer Detection
    • Built a CNN-based classifier to categorize wafer states (Normal, UnCMP, OverCMP, Cleaning Error).
    • Integrated Class Activation Mapping (CAM) to visualize and localize defective regions, providing actionable insights for engineers.
  • Sensor Anomaly Detection
    • Designed an LSTM Encoder-Decoder architecture to reconstruct sensor data streams.
